## Sobre el proyecto

**Taibuddy** es una aplicación web pensada para la preparación de oposiciones (exámenes de acceso a la función pública en España). La idea es ofrecer una herramienta de estudio con tests, seguimiento de progreso y estadísticas.

## Motivación

Preparar oposiciones implica estudiar grandes volúmenes de temario y hacer cientos de tests tipo. Las herramientas existentes son caras o poco flexibles. Taibuddy pretende ser una alternativa accesible y adaptable.

## Stack técnico

- **Backend**: Laravel con Livewire para interactividad
- **Base de datos**: MySQL
- **Autenticación**: Sistema de usuarios con roles

## Planes futuros

- Hacer la plataforma **genérica** para que sirva para cualquier tipo de examen, no solo oposiciones
- Sistema de importación de bancos de preguntas
- Estadísticas avanzadas y modo repaso inteligente

## Sobre el proyecto

**Liceo de Sevilla** es el portal web integral de una asociación musical de Sevilla. Nació como una colaboración para dar presencia digital a la asociación, evolucionando hacia un CMS robusto para gestionar noticias, álbumes, repertorios, hermandades y procesiones.

## Stack técnico

- **Backend / Core**: Laravel 11 (PHP 8.2+)
- **Panel de Administración**: Filament Admin Panel 3.x (sistema completo de recursos y CRUD)
- **Base de datos**: MySQL / PostgreSQL
- **Frontend Público**: Blade templates + TailwindCSS
- **Despliegue automatizado**: GitHub Actions con scripts en PHP (`deploy.php`) orientados a un hosting compartido sin `proc_open()`

## Características clave

- **Sitio Público**: Rutas públicas y dinámicas manejadas desde Laravel (configuración en `menu.php`).
- **CMS Autogestionable**: Gestión integral bajo `/admin` manejada mediante los Componentes Visuales y Form Builder de Filament, protegida por autenticación.
- **Sitemap Dinámico**: Generación automatizada mediante CRON para mejor SEO.

## Mi rol

Participé de forma colaborativa en el desarrollo, destacando en:
- Diseño y construcción de la arquitectura MVC con Laravel.
- Configuración y personalización del Panel de Filament para dotar a la asociación de independencia en el manejo de su contenido.
- Implementación de flujos de CI/CD para automatizar subidas al servidor de producción vía FTP con GitHub Actions.

## About the project

**Equill** (codename: *killbill*) is an application for managing shared expenses between couples. The core idea is simple: track who pays what and offer flexible ways to settle up.

## How it works

1. Both members log their shared expenses
2. The app calculates who owes whom
3. Settlement can happen in several ways:
   - **Bank transfer** (Bizum in Spain)
   - **Cash**
   - **Compensation**: pay for the other's expense to balance things out

## Tech stack

- **Backend**: Laravel with REST API
- **Database**: MySQL
- **Frontend**: In planning phase

## Future plans

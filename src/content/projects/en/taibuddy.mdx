## About the project

**Taibuddy** is a web application designed for preparing *oposiciones* (civil service entrance exams in Spain). It aims to provide a study tool with practice tests, progress tracking, and statistics.

## Motivation

Preparing for civil service exams involves studying large volumes of material and completing hundreds of practice tests. Existing tools are expensive or inflexible. Taibuddy aims to be an accessible and adaptable alternative.

## Tech stack

- **Backend**: Laravel with Livewire for interactivity
- **Database**: MySQL
- **Authentication**: User system with roles

## Future plans

- Make the platform **generic** so it can serve any type of exam, not just civil service
- Question bank import system
- Advanced statistics and smart review mode

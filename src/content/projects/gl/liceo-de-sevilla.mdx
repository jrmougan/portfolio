## Sobre o proxecto

**Liceo de Sevilla** é o portal web integral dunha asociación musical de Sevilla. Naceu como unha colaboración para dar presenza dixital á asociación, evolucionando cara a un CMS robusto para xestionar novas, álbums, repertorios, irmandades e procesións.

## Stack técnico

- **Backend / Core**: Laravel 11 (PHP 8.2+)
- **Panel de Administración**: Filament Admin Panel 3.x (sistema completo de recursos e CRUD)
- **Base de datos**: MySQL / PostgreSQL
- **Frontend Público**: Blade templates + TailwindCSS
- **Despregue automatizado**: GitHub Actions con scripts en PHP (`deploy.php`) orientados a un hosting compartido sen recursos de consola (`proc_open`).

## Características clave

- **Sitio Público**: Rutas públicas e dinámicas manexadas desde Laravel (`menu.php`).
- **CMS Autoxestionable**: Xestión integral baixo `/admin` manexada cos Compoñentes Visuais e Form Builder de Filament, protexida por autenticación.
- **Sitemap Dinámico**: Xeración automatizada mediante CRON para mellorar o SEO.

## O meu rol

Participei de forma colaborativa no desenvolvemento, destacando en:
- Deseño e construción da arquitectura MVC con Laravel.
- Configuración e personalización do Panel de Filament para dotar á asociación de independencia no manexo do seu contido.
- Implementación de fluxos de CI/CD para automatizar subidas ao servidor de produción vía FTP con GitHub Actions.

## Estado

O proxecto está **en produción** e activo en [liceodesevilla.com](https://liceodesevilla.com).
